# 💳 Python ATM Management System

A simple yet powerful **ATM Simulation System** built using Python.
This project demonstrates core programming concepts like **dictionaries, modular coding, authentication, and transaction handling**.

---

## 🚀 Features

### 👤 User Features

* Secure Login System (PIN-based) 
* Check Account Balance 
* Deposit Money with Timestamp
* Withdraw Money with Daily Limit (₹10,000/day) 
* Transaction History Tracking
* Mini Dashboard with analytics 📊 

---

### 🔐 Admin Features

* Admin Login System 
* Add New Users
* Remove Existing Users
* View All Users and Balances

---

## 🧠 Concepts Used

* Python Dictionaries (Data Storage)
* Modular Programming (Multiple Files)
* Loops & Conditional Statements
* Date & Time Handling
* Basic Security (PIN Authentication)

---

## 📂 Project Structure

```bash
ATM/

```

## ⚙️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/aarushdevworld/basic-atm.git
```

2. Navigate to project:

```bash
cd basic-atm
```

3. Run the program:

```bash
python main.py
```
